Why this exists

The ecosystem has the taxonomy (OWASP list) and scanners (Promptfoo, Garak, PyRIT) but few minimal, runnable attack→defense pairs a practitioner can read in 15 minutes and run in 60 seconds. This fills that gap.

Teaching with this repo

Instructors: use this repo in your security training. Each category is self-contained and follows the same structure. See the Teaching Guide for tips on using these demos in the classroom.
